## System Startup (Step-by-Step)

### Step 1 — Install Python dependencies
```bash
cd fossbot_sim/python
pip install -r requirements.txt
```

### Step 2 — Start the relay server
```bash
python server.py --host 0.0.0.0 --port 5000
# Visit http://localhost:5000 to confirm it's running
```

### Step 3 — Start Godot
- Open your Godot project.
- In `FossHandler.gd`, the SocketIO node connects to the server with `?role=godot`.
- Ensure `SimInfo` is registered as AutoLoad.
- Press **Play** — the server dashboard will show `Godot: ✅ Connected`.

### Step 4a — Run the demo
```bash
# Write example scenario first (only needed once)
python demo_navigation.py --write-scenario --scenario scenarios/example_maze.yaml

# Run full A* + DWA navigation demo
python demo_navigation.py --scenario scenarios/example_maze.yaml
```

### Step 4b — Train with Stable-Baselines3
```bash
python train_sb3.py --algo ppo --scenario scenarios/example_maze.yaml --steps 500000
# Evaluate
python train_sb3.py --eval \
    --model logs/fossbot_ppo_final \
    --vecnorm logs/fossbot_ppo_final_vecnorm.pkl
```

### Step 4c — Use gym.make()
```python
import fossbot_gym_register   # registers FossBot-v0, FossBot-Maze-v0, etc.
import gymnasium as gym

env = gym.make("FossBot-Maze-v0")
obs, info = env.reset()
for _ in range(100):
    action = env.action_space.sample()
    obs, reward, terminated, truncated, info = env.step(action)
    if terminated or truncated:
        obs, info = env.reset()
env.close()
```

## Costmap-Aware A* Example

```python
import heapq
from fossbot_client import FossBotClient

def astar_with_costmap(start, goal, costmap):
    """Cost-Aware A* using terrain traversal costmap (Chapter 9.1.3)."""
    open_set = [(0, start)]
    came_from = {}
    g_score = {start: 0.0}

    while open_set:
        _, current = heapq.heappop(open_set)
        if current == goal:
            path = []
            while current in came_from:
                path.append(current)
                current = came_from[current]
            return list(reversed(path))

        for dx, dz in [(1,0),(-1,0),(0,1),(0,-1)]:
            neighbor = (current[0]+dx, current[1]+dz)
            terrain_cost = costmap.get(neighbor, 1.0)
            tentative_g = g_score[current] + terrain_cost
            if neighbor not in g_score or tentative_g < g_score[neighbor]:
                came_from[neighbor] = current
                g_score[neighbor] = tentative_g
                h = abs(goal[0]-neighbor[0]) + abs(goal[1]-neighbor[1])
                heapq.heappush(open_set, (tentative_g + h, neighbor))
    return []

with FossBotClient() as client:
    client.load_scenario_file("scenarios/example_maze.yaml")
    costmap = client.get_costmap_grid()

    path = astar_with_costmap((0,0), (8,8), costmap)
    # Convert to 3D coords and visualise
    path_3d = [[x, 0.1, z] for x, z in path]
    client.set_planned_path(path_3d)
    client.set_state_label("Planning")
```

## Noise Configuration

Override noise per-scenario in the YAML file:
```yaml
noise_config:
  ultrasonic:
    enabled: true
    stddev: 0.05       # higher = noisier sonar
  ir:
    enabled: true
    prob: 0.08         # 8% dropout
  imu:
    enabled: false     # clean IMU for this scenario
```

Or programmatically:
```python
from fossbot_client import FossBotClient
client.load_scenario(scenario)  # scenario.noise_config overrides SimInfo
```
